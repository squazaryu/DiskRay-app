## DRay 2.0.3

Release date: 2026-04-21

### Highlights
- Stage-based maintainability pass completed (no product-direction rewrite):
  - Root: responsibility concentration reduced with focused collaborators under `DRay/App/Root/`
  - Performance: workspace/presentation decomposition completed without IA/UX redesign
  - Settings: section decomposition + clearer permission impact mapping grounded in real gate usage
