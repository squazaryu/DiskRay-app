## DRay 2.0.2

### Highlights
- Focused cleanup pass for performance hygiene and state propagation.
- Reduced repeated system command load in energy consumers collection with short-lived caching and in-flight request coalescing.
- Coalesced post-mutation rescans to avoid redundant heavy refreshes after trash/restore operations.
- Reduced root-level UI fan-out by removing global feature `objectWillChange` forwarding.
- Extracted workspace side effects (`open` / `reveal`) from root glue into `WorkspaceActionService`.
- Reduced helper idle work by disabling expensive consumer sampling while popup is hidden.
- Added deterministic tests for `EnergyConsumersService` caching and request coalescing behavior.
