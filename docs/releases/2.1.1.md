## DRay 2.1.1

Release date: 2026-05-08
Build: 1

### Highlights
- Menu bar identity correction:
  - default label is now DRay-first (`DRay mark + battery state`) instead of battery-only
  - charging and low-battery states remain visible in compact form
- Popup structure parity:
  - one shared light/dark popup structure with consistent block order and actions
  - telemetry footer is preserved with uptime and network transfer rates
- Adaptive sidebar:
  - added `Adaptive / Expanded / Collapsed` modes
  - adaptive rule expands sidebar on wide windows and collapses on compact windows
- Search/Uninstaller deletion hardening:
  - better permission diagnostics when Full Disk Access exists but direct delete is denied
  - optional Experimental Full Access path for administrator-authorized Trash moves
  - SIP/TCC-protected paths remain explicitly blocked and reported

### Changed
- `RootView` decomposed into shell components (`RootSidebarView`, `RootPermissionOnboardingCard`, `RootSectionRouter`) without changing ownership in `RootViewModel`.
- `SettingsView` reduced to shell composition; status strip extracted and accessibility for Launch at Login toggle corrected.
- Added lifecycle audit notes for `LiveSystemMetricsMonitor` and Uninstaller smoke/regression notes in `docs/`.
