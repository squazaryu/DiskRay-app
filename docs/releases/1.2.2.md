# DRay 1.2.2

Release date: 2026-04-12

## Highlights
- Performance module extended with on-demand network speed testing via macOS `networkQuality`:
  - run-only when requested (no background load)
  - download/upload throughput + responsiveness + base RTT
  - explicit separation from live network counters
- Live network formatter improved to avoid false “Zero KB/s” for low traffic.
- Tests added for network speed report assembly and controller orchestration.

## Included Functionality Snapshot
- Smart Care, My Clutter, Uninstaller, App Repair, Space Lens, Search, Performance, Privacy, Recovery, Settings.
- Independent menu-bar helper with live metrics and app handoff actions.
- Recovery history and rollback support for DRay-managed operations.
