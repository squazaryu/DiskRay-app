# DRay 1.2.1

Release date: 2026-04-12

## Highlights
- Performance module extended with battery and energy analytics (inside main app only):
  - device-level battery summary (charge, health, cycles, state, draw, ETA, temperature)
  - top process energy consumers with transparent per-metric breakdown
  - honest per-process estimate: **Estimated Drain Share** (explicitly marked as estimate)
- Architecture iteration continued:
  - Smart Care orchestration moved into `SmartCareFeatureController`
  - My Clutter / duplicates orchestration moved into `DuplicatesFeatureController`
  - additional RootViewModel responsibility reduction with feature-local state/flows
- Stability and UX fixes:
  - Performance screen layout restored (header visibility + vertical scrolling)
  - test coverage expanded for new controllers and battery/energy report assembly

## Included Functionality Snapshot
- Smart Care, My Clutter, Uninstaller, App Repair, Space Lens, Search, Performance, Privacy, Recovery, Settings.
- Independent menu-bar helper with live metrics and app handoff actions.
- Recovery history and rollback support for DRay-managed operations.
