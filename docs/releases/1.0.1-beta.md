# DRay 1.0.1-beta

Release date: 2026-04-03

## Highlights
- Final polish fixes for this beta:
  - `Health details` popup now uses full available width (no compressed alert row)
  - `Performance` module content is top-aligned inside the inner shell (no vertical centering drift)
  - Popup `Reduce CPU/Reduce Memory` flow stabilized: inline confirm + inline result banner, no popup auto-close and no forced navigation to main window
- Fixed and stabilized icon/theming pipeline:
  - regenerated icon assets (`DRay.icns`, `DRayLight.icns`, `DRayDark.icns`)
  - improved icon canvas usage and removed compressed inner panel effect
  - helper now auto-syncs `/Applications/DRay.app` icon to current system appearance
- Improved menu-helper reliability:
  - added robust appearance-change observer in helper process
  - fixed early app-appearance access crash path
- Product-level GitHub repository refresh:
  - rewritten `README` focused on DRay capabilities and module map
  - updated release runbook (`docs/RELEASE.md`) for beta pipeline

## Included Functionality Snapshot
- Smart Care + My Clutter + Uninstaller + App Repair + Space Lens + Search + Performance + Privacy + Recovery + Settings.
- Independent menu-bar helper popup with live telemetry and quick actions.
- Recovery history for DRay-managed deletions.
- Permissions-aware scan/cleanup model compliant with macOS security restrictions.
