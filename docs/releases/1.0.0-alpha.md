# DRay 1.0.0-alpha (Pre-release)

Release date: 2026-03-31

## Highlights
- Unified liquid-glass shell and refreshed module headers across Smart Care, My Clutter, Uninstaller, Space Lens, Performance, and App Repair.
- Stable menu-bar helper runtime split (`DRayMenuBarHelper`) with startup/open-section/quit-handoff smoke coverage.
- Expanded battery diagnostics in popup:
  - history windows `5m/15m/30m/1h/2h`
  - trend sparkline
  - rate + ETA + range summary
- Performance module upgrades:
  - sustained trends (`5m/15m`)
  - live consumer panel with normalized app names
  - in-app process de-prioritization and priority restore
- Uninstaller/App Repair flow:
  - app icons in list
  - artifact risk labels
  - rollback sessions
  - `Safe Reset` / `Deep Reset` strategy with Deep confirmation and preview

## Packaging
- Artifacts:

  - `DRay-1.0.0-alpha.dmg`

## Notes / Limitations
- This is an alpha build.
- Full Disk Access and folder access are required for complete scan/cleanup coverage.
- Not all modules have final production UI polish yet; functional parity work continues.
