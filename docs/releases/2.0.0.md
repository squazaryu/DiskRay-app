## DRay 2.0.0

### Highlights
- Premium redesign shell with adaptive sidebar (auto icon-only mode on narrow windows) and cleaner section hierarchy.
- Performance workspace reorganized into focused zones: Overview, System Load, Battery & Energy, Startup & Network.
- Menu bar helper popup redesigned as a compact companion panel with stronger action hierarchy and clearer health summary.
- Search/Smart Care/App Repair/Space Lens readability improvements for medium and narrow window sizes.
- On-demand network speed test flow (no passive network polling).

### UX and Stability
- Removed duplicated global section-header action buttons from app shell.
- Smart Care content now anchors to top and avoids mid-screen drift on large windows.
- Sidebar width tuned to free canvas area for Space Lens and data-heavy modules.
- Updated icon pipeline and regenerated light/dark `.icns` assets for v2 visual identity.

### Notes
- DiskRay estimates (for example, `Estimated Drain Share`) are explicitly labeled as estimates and are not official per-app battery percentages from macOS.
- Full Disk Access is still required for complete system-level cleanup coverage on protected paths.
