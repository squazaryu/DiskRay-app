## DRay 2.1.0

Release date: 2026-05-01
Build: 1

### Highlights
- New Overview dashboard:
  - live health status, CPU, memory, battery, and storage tiles
  - recommendations, top live consumers, recent activity, and bottom status strip
- Refreshed 2.1 UI system:
  - collapsed icon navigation with Overview as the default landing section
  - shared infographic components, sparklines, ranked rows, progress bars, and status rings
  - adaptive interface density plus persisted accent-color preferences
  - updated app icon assets for light/dark variants
- Uninstaller upgrades:
  - Remaining workspace for unresolved artifacts after uninstall/verify
  - Deep Sweep for orphaned Library/startup remnants from apps removed outside DRay
  - persisted observed-app and remaining-artifact records
  - clearer completion feedback for Remaining scans and cleanup actions
- Feature workspace polish across Smart Care, My Clutter, Space Lens, Search, Performance, Privacy, Recovery, Repair, Settings, and the menu-bar helper.
- Menu bar helper redesign:
  - compact health hero, metric tiles, sparklines, top consumers, and quick actions
  - battery details lifecycle and heavy-sampling refresh discipline preserved

### Changed
- `RootViewModel` remains the app coordinator; new UI settings are persisted through `UISettingsStore`.
- `Performance` keeps session-scoped trend/history visuals.
- `Settings` now separates general controls from appearance controls and exposes accent color and interface density.
- `Uninstaller` now stores Remaining and observed-app metadata under the operational history store.
