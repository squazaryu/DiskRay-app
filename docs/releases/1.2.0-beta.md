# DRay 1.2.0-beta

Release date: 2026-04-10

## Highlights
- Architecture decomposition milestone (`M13`) in active code:
  - root shell/orchestration boundary tightened
  - feature-controller extraction introduced for Search, Privacy, Recovery, Uninstaller, Repair, and Performance
  - persistence contracts expanded to reduce direct feature writes to raw storage
- Search and cleanup workflows improved:
  - search scope handling prepared for startup-disk and selected-target flows
  - bulk selection and cleanup UX in Search stabilized
  - safer batch-trash handling for parent/child path overlaps
- Menu-bar helper stabilization:
  - battery details now opened from explicit `Details` action (instead of fragile card edge toggle)
  - close behavior hardened for popup lifecycle transitions
  - helper telemetry publishing reduced to lower UI/window churn
- Uninstaller and permission-gate reliability updates:
  - better handling/reporting for protected/system-managed artifacts
  - improved diagnostics for partial-removal scenarios

## Included Functionality Snapshot
- Smart Care, My Clutter, Uninstaller, App Repair, Space Lens, Search, Performance, Privacy, Recovery, Settings.
- Independent menu-bar helper with live metrics and direct module handoff actions.
- Recovery history and rollback support for DRay-managed operations.
