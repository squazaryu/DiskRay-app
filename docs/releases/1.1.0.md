# DRay 1.1.0

Release date: 2026-04-06

## Highlights
- Architecture stabilization pass:
  - extracted feature-level use-cases for Smart Care, Performance, Search, Recovery, and Uninstaller flows
  - split large root orchestration into feature view models and clearer dependencies
  - introduced safe file-operation coordination for protected-path checks and permission-aware actions
- Operational resilience improvements:
  - quick-action delta reports for before/after bytes and items
  - rollback snapshots for quick-clean batches in Recovery
  - operational history storage moved to SQLite with migration from legacy storage
- Quality and release hardening:
  - expanded deterministic unit tests for permission gates, uninstall verify, incremental scan merge, presets/history use-cases
  - integrated PII pre-release scan and sanitized docs/examples

## Included Functionality Snapshot
- Smart Care, My Clutter, Uninstaller, App Repair, Space Lens, Search, Performance, Privacy, Recovery, Settings.
- Independent menu-bar helper with live telemetry and quick actions.
- Recovery history and rollback support for DRay-managed actions.
